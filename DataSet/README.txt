# Exported Dataset from Rakib / Motion Detector

This repository contains the dataset exported from the Rakib / Motion Detector project. Use this dataset to train and validate machine learning models in Edge Impulse.

## How to Import the Dataset

### Using the Edge Impulse CLI

1. **Install the Edge Impulse CLI:**  
   If you haven't installed the CLI yet, follow the instructions in the [Edge Impulse CLI Overview](https://docs.edgeimpulse.com/docs/edge-impulse-cli/cli-overview).

2. **Upload the Dataset:**  
   Open your terminal and run the following command to import the dataset into a new Edge Impulse project:

   ```bash
   edge-impulse-uploader --clean --info-file info.labels
